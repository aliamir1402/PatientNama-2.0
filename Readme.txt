For data storage and management, I integrated PostgreSQL, a powerful open-source relational database. PostgreSQL provided a reliable and scalable solution for storing and retrieving data. Throughout the project, I collaborated closely with the development team, participating in agile methodologies such as daily stand-ups, sprint planning, and code reviews.
